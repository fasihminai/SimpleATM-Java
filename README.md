# SimpleATM-Java
Simple ATM project in Java demonstrating OOP principles


A simple **console-based ATM program** written in Java demonstrating **Object-Oriented Programming (OOP) concepts** such as Encapsulation, Abstraction, Inheritance, and Polymorphism.

---

## Features

- **Login with PIN** (3 attempts max)  
- **Check Account Balance**  
- **Deposit Money**  
- **Withdraw Money**  
- **Exit Program**  
- **Simple console-based interface** â€” easy to use  

---

## OOP Concept Used

### Encapsulation
- PIN and balance are private variables
- Access is controlled through public methods

---

## Usage

1. Enter your PIN (default: 1234).  
2. If wrong PIN is entered 3 times, the program will block access.  
3. Choose from the menu to check balance, deposit, withdraw, or exit.

## How to Run

1. Clone the repository:
```bash
git clone https://github.com/fasihminai/SimpleATM-Java.git
```
2. Navigate to the project folder:
```bash
cd SimpleATM-Java
```
3. Compile the Java file:
```bash
javac MyATMProject1.java
```
4. Run the program:
```bash
java MyATMProject1
```
